# unr's Oni Configs

Keeping track of my customization to Oni.

## Install my configs

```
cd ~/.config
g clone --recurse-submodules git@github.com:unr/oni.git
plugins/fzf/install
```


## Plugin List

> Installed via submodule.

- [fzf](https://github.com/junegunn/fzf)
- [fzf.vim](https://github.com/junegunn/fzf.vim)
